# Pedro H. P. Rocha — Academic Website

Jekyll site for GitHub Pages.

## Files to edit

- `_config.yml` — your name, email, site URL
- `index.md` — About page content
- `pages/cv.md` — CV page (replace PDF at `assets/files/cv.pdf`)
- `pages/teaching.md` — Teaching page
- `pages/contact.md` — Contact info
- `_posts/` — Blog posts (format: `YYYY-MM-DD-title.md`)
